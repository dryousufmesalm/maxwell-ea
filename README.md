# Maxwell EA

Client-style EA with dashboard and box UI (Dashboard.mqh and Box.mqh included in this repo).

**Author:** Yousuf Mesalm | www.yousufmesalm.com | WhatsApp +201006179048 | Upwork: https://www.upwork.com/freelancers/youssefmesalm

## Requirements

- MetaTrader 5
- YM library in MQL5/Include/YM

## Installation

Copy all files from this repo into MQL5/Experts/ or set Include path so the compiler finds them.

## Compilation

Compile MaxwellEA.mq5 in MetaEditor (F7). Include path must include the terminal MQL5 folder.

## Usage

Attach to a chart. Use the dashboard/box controls as designed.
